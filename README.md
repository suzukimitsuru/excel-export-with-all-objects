# excel-extructor-with-all-objects

Microsoft Excel の差分を見る為のツールです。
図形や画像も含めて比較できるツールが少ないため、作成しました。

## Loadmap

- v0.0.1 テキストの抽出 → 今ココ！
    - セルのテキストを抽出する
    - 図形のテキストを抽出する
- v0.0.2 修飾の抽出
    - セルのテキストの修飾を抽出する
    - 図形のテキストの修飾を抽出する
